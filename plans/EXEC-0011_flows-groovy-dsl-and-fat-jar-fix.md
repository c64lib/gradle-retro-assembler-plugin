Root cause not chased down (Groovy nested-closure argument-expression delegate resolution); narrow enough to be a separate follow-up rather than a PLAN-0011 blocker. Candidate for a new issue if Groovy consumers are expected to use `useFrom()`/`useTo()` DSL sugar.
